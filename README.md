## Project Description
This project is a prototype design for an AI-powered Forex Trading Analysis Bot.
It automates technical and fundamental analysis and provides explainable,
multi-timeframe directional suggestions to support Forex traders in decision making.

## Features
- Multi-timeframe technical analysis (SMA, EMA, RSI, MACD)
- AI-assisted directional market analysis
- Explainable AI output
- User-customizable indicators
- Simple web-based interface

## Tools & Technologies
- Python (FastAPI / Flask)
- Pandas, NumPy, scikit-learn
- React / JavaScript
- Forex APIs (OANDA, Alpha Vantage)
